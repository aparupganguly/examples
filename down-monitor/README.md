# hb-uptime

Real-browser uptime checker. Not just HTTP pings.

## Get an API Key

Get your API key at **[https://hyperbrowser.ai](https://hyperbrowser.ai)**

```bash
export HYPERBROWSER_API_KEY=your_api_key_here
```

## Usage

```bash
npm install
npm run build
npm start -- https://example.com
```

Optional: Analyze outages with Gemini 3.0 Pro preview

```bash
export GEMINI_API_KEY=your_gemini_key
npm start -- --analyze https://example.com
```

Follow @hyperbrowser for updates.
